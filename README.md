# 🚀 EnterpriseERP

EnterpriseERP is a modern Multi-Tenant Enterprise Resource Planning (ERP) system built using ASP.NET Core 8, Angular, SQL Server, and Clean Architecture.

## 🎯 Project Goal

The goal of this project is to build a production-ready ERP system by following enterprise software development practices instead of creating a simple CRUD application.

This project is being built publicly to demonstrate:

- Clean Architecture
- Multi-Tenant Architecture
- ASP.NET Core 8
- Angular
- Entity Framework Core
- SQL Server
- JWT Authentication
- Role-Based Access Control (RBAC)
- Repository Pattern
- Unit of Work
- REST APIs
- Docker
- Azure Deployment
